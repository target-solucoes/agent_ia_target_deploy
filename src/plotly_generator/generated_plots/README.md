# Generated Plots Directory

Este diretorio contem graficos gerados pelo Plotly Generator Agent.

## Formatos de Saida

- **HTML**: Graficos interativos (default)
- **PNG**: Graficos estaticos (requer kaleido)

## Nomenclatura

Arquivos gerados seguem o padrao:
```
chart_YYYYMMDD_HHMMSS.{html|png}
```

Exemplo: `chart_20251112_143022.html`

## Limpeza

Arquivos neste diretorio sao temporarios e podem ser removidos a qualquer momento.
O diretorio e automaticamente recriado quando necessario.
